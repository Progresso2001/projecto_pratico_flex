Minha Trajectória na Porgramação

Saudações, pessoal! O meu nome é Joaquim Eliseu Menianga, e hoje quero partilhar convosco a minha jornada na programação – uma aventura que começou com curiosidade e hoje me leva a sonhar em criar projetos incríveis. Tudo começou há alguns anos, quando eu era só um estudante de Ciência da Computação na Rússia (Tula), fascinado por como o código podia transformar ideias em realidade. No início, era HTML e CSS: criei as minhas primeiras páginas web simples, brincando com layouts, posicionamento e responsividade. Lembro-me de noites inteiras no Visual Studio Code, ajustando cores e fazendo sites que pareciam mágicos no telemóvel. Era o frontend a conquistar-me! Mas o backend chamou-me rápido. Mergulhei no Python – a linguagem perfeita para um iniciante como eu. Aprendi lógica de programação, algoritmos básicos e comecei a construir APIs com Flask e FastAPI. Integrações com bases de dados como MySQL, SQLite e SQLAlchemy foram o próximo passo: criei projetos com operações CRUD, gerindo dados como um pro. Git e GitHub tornaram-se os meus melhores amigos – branching, commits e repositórios no GitHub Codespaces ajudaram-me a versionar tudo direitinho.

Hoje, exploro mais: visualização de dados com Matplotlib e Pandas, automação com ferramentas como N8N, e até UI/UX no Figma e Canva. Testes unitários, deployment em Heroku ou Docker, e clean code são as minhas obsessões. Uso Postman e Apidog para debugar APIs, e o Chrome DevTools para frontend afiado. São projetos pessoais, homework e rotinas diárias de estudo no Notion que me moldaram. Esta trajetória não é linear – tem bugs, frustrações e "eurekas" épicos. Mas cada linha de código me aproxima do sonho: ser um developer júnior sólido, criar conteúdo no YouTube sobre rotinas de dev e impactar a comunidade angolana. Se estás a começar, começa pequeno, pratica diariamente e junta-te a mim nesta viagem!

Videos aula e sites de Programação

Aqui estão alguns vídeos-aulas e sites que foram fundamentais para progredir na programação, especialmente para iniciantes como tu, focados em conteúdos gratuitos em português e alinhados às tuas áreas de interesse (Python, web dev, Flask e dados).

Canais YouTube Principais

  • Curso em Vídeo (Gustavo Guanabara): Aulas completas e didáticas sobre HTML, CSS, JavaScript, Python e lógica de programação. Começa do zero com playlists organizadas – perfeito para frontend e backend básico.
  • Programação Dinâmica: Dicas práticas para iniciantes, como "3 DICAS para APRENDER a PROGRAMAR do ZERO", com foco em motivação, lógica e projetos reais.
  • Hastag Programação: Lives como "GUIA COMPLETO para Programadores Iniciantes para Python" cobrem escolha de linguagens, front/back-end e primeiros projetos

Técnicas Eficazes

  • Pomodoro adaptado: 25min código + 5min stretch; após 4 ciclos, 15min de Postman para testar APIs.
  • Projetos semanais: Semana 1: app web simples (HTML/CSS/JS); Semana 2: backend Python com banco MySQL.
  • Debug diário: 15min no Chrome DevTools ou Apidog para fixar erros comuns.
