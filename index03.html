Introdução ao HTML

O HTML é a linguagem de marcação fundamental para criar a estrutura de páginas web. Como estudante de Ciência da Computação com foco em desenvolvimento web, você pode começar criando arquivos simples para praticar.

Significado HTML

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto). É uma linguagem de marcação, não de programação, usada para definir a estrutura semântica de conteúdos como títulos, parágrafos, imagens e links. Ela organiza o conteúdo para que navegadores o renderizem corretamente.

Elementos Essenciais

  • Cabeçalhos: h1 a h6 para títulos hierárquicos
  • Parágrafos: p para textos.
  • Links: usando uma ancora href="https://exemplo.com" Texto para hiperlinks.
  • Imagens: img src="imagem.jpg" alt="Descrição" para fotos acessíveis.
  • Listas: ul para não ordenadas e ol para numeradas, com li para itens.

Use semântica moderna como header, main, section e footer para melhor acessibilidade e SEO.
